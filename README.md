# OctoFish — Система управления рыболовецким флотом

## Описание проекта

OctoFish представляет собой веб-приложение для централизованного управления малым и средним рыболовецким флотом. Система объединяет функции мониторинга судов, учета улова по видам рыб, регистрации рейсов, подачи и обработки заявок, а также визуализации промысловых данных на интерактивной карте.

## Функциональные возможности

### Просмотр данных
- Список судов с характеристиками, статусом и детализацией улова
- Таблица рейсов с раскрывающимися деталями
- Интерактивная карта мест лова с цветовой индикацией уловистости
- Дашборд статистики на главной странице

### Авторизация и роли
- Гость: просмотр судов, рейсов, карты и статистики без входа
- Клиент: подача заявок на суда, рейсы и места лова после авторизации
- Администратор: полный доступ, обработка заявок, редактирование данных

### Управление рейсами
- Создание рейса с выбором судна, места лова и капитана
- Добавление улова по видам рыб с контролем вместимости судна
- Прогресс рейса рассчитывается от водоизмещения судна
- Завершение рейса с переносом улова в статистику судна

### Обработка заявок
- Подача заявок трех типов: судно, рейс, место лова
- Одобрение заявок администратором с автоматическим добавлением в систему
- Отклонение заявок с сохранением истории

## Технологии

| Компонент | Технология |
|-----------|------------|
| Frontend | React, TypeScript, Material UI, OpenLayers, Vite |
| Backend | Python, FastAPI, SQLAlchemy, SQLite |
| Аутентификация | JWT-токены, bcrypt |
| Карты | OpenLayers (Яндекс, Google, OSM) |
| Хранение | LocalStorage (клиент), SQLite (сервер) |

## Установка и запуск

cd backend
pip install -r requirements.txt
python init_demo.py
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

cd frontend
npm install
npm run dev
